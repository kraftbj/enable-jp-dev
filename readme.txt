== Description ==

Jetpack automatically enables a connection-free Development Mode under a few situations. Sometimes, you need Development Mode other times. This is for you.

== Installation ==

1. Install the plugin within WordPress via the Plugins->Add New page.
2. Activate Plugin
3. Profit. No options to set, nothing else to do.


== Frequently Asked Questions ==

= Why would I want to use this? =

You're using a staging site. You're developing on a local server and using a HOSTS file hack to avoid changing URLs.
